# Smart AI Failover

Fail-safe AI request platform built with Next.js, TypeScript and Supabase.  
Primary provider: Gemini  
Fallback provider: Claude

---

## 🚀 Features

- Automatic AI failover (Gemini → Claude)
- Zero downtime response flow
- Request history dashboard
- Supabase logs storage
- Real-time updates
- Loading / error states
- Clean SaaS UI
- Docker support

---

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase
- Vitest
- Docker
